推荐服务费支付时间

星火于每月10日前核对上个月（自然月）的结算账目、经系统确认无误后，每月20日支付至理财师在星火绑定的银行账户。 (如遇节假日和法定休息日顺延至下一工作日支付，结算金额不足100元，累计至下一期结算)。

推荐服务费计算方式

产品系列 推荐服务费计算公式
零投宝

推荐服务费=推荐服务费基数*费率*费率系数

推荐服务费基数=交易金额*（1/365）*出借天数
月盈宝

每月推荐服务费=推荐付费基数*费率*费率系数

推荐服务费基数=每期出借资金 *（1/365）*每期出借天数
定制产品

推荐服务费=推荐服务费基数*费率*费率系数

推荐服务费基数=交易金额*（1/365）*出借天数
星系列

推荐服务费=推荐服务费基数*费率*费率系数

推荐服务费基数=交易金额*（1/12）*投资期限（月）
自选标

每日推荐服务费=推荐服务费基数*费率*费率系数

推荐服务费基数=每日持有的出借本金（1/365）*1
  • 客户交易金额：是指客户单笔成功支付的出借资金
  • 客户出借天数：自出借人通过支付机构成功支付出借资金的次日（工作日）至用户出借资金到期日的前一日，即客户出借资金的整个锁定期间。
  • 费率：若存在浮动费率，则以基础费率+浮动费率计算。

推荐服务费扣税说明

根据国家规定，推荐服务费总额超过800时需扣除个人劳务所得税。具体扣税公式如下：

星火当月推荐服务费 星火扣税计算方法
当月推荐服务费<=800时 扣税额 = 0
800<当月推荐服务费<=4000时 扣税额 = （收入-800）×20% - 0
4000<当月推荐服务费<=20000时 扣税额 = 收入×（1-20%）×20% - 0
20000<当月推荐服务费<=103000时 应纳税所得额 = 收入×（1-20%）
应纳税所得额<=20000 扣税额 = 应纳税所得额×20% - 0
20000<应纳税所得额<=50000 扣税额 = 应纳税所得额×30% - 2000
应纳税所得额>50000 扣税额 = 应纳税所得额×40% - 7000
103000<当月推荐服务费时 增值税 = 当月推荐服务费/1.03×3%
城建税 = 增值税×7%
当月推荐服务费/1.03<=100000 教育费附加&地方教育费附加 = 0
当月推荐服务费/1.03>100000 教育费附加&地方教育费附加= 增值税×5%
扣除增值税及附加后收入 = 当月推荐服务费 – （增值税+城建税+教育费附加&地方教育费附加）
应纳税所得额 = 扣除增值税及附加后收入×（1-20%）
20000<应纳税所得额<=50000 所得税 = 应纳税所得额×30% - 2000
50000<应纳税所得额 所得税 = 应纳税所得额×40% - 7000
扣税额 = 增值税及附加+所得税
